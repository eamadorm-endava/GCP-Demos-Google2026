# Agentic Vendor Governance Platform - Live Demo Script

**About The Demo**
This demo illustrates how **Gemini Enterprise** empowers procurement organizations to build next-generation vendor management that transcends static dashboards and manual reporting. Instead of relying on vendors to "grade their own homework," we showcase an **Autonomous Governance Agent** that moves from *Descriptive Analytics* (what happened) to *Prescriptive AI* (what to do about it).
